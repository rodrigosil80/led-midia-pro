# 📺 LED Mídia Pro

Sistema profissional para gerenciamento de publicidade em painéis de LED.

## 🚀 Tecnologias

- Python
- CustomTkinter
- SQLite
- OpenPyXL
- Git
- GitHub

## 🎯 Objetivo

Desenvolver um sistema moderno para gerenciamento de anúncios em painéis de LED, permitindo:

- Cadastro de clientes
- Dashboard administrativo
- Relatórios
- Exportação para Excel
- Exportação para PDF
- Controle financeiro
- Estatísticas do painel
